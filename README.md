Hike More is inspired by the website [AllTrails](https://www.alltrails.com/) with an emphasis on users being able to create and search for hikes within the database

## Technologies
- React/Redux
- Python
- Flask
- SQLAlchemy
- PostgresQL
- GoogleMaps API
